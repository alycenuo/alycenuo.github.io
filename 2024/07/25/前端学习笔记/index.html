前端学习笔记

Created2024-07-25|Updated2024-11-09

如果主轴为水平方向，那么交叉轴为垂直方向，之所以称为交叉轴是因为当主轴设定为垂直方向时，交叉轴为水平方向。两者是垂直关系。

当主轴设定为水平方向时，内容就像对联或者说像日文小说的文字一样是从上到下排列的。此外，跟主轴和交叉轴相关的概念还有“起始线和终止线”，例如文字是从左往右还是从右往左，不同的语言模式下是不同的。

主轴和起始线都是通过flex-direction设定。

文档中采用了 flexbox 的区域就叫做 flex 容器。为了创建 flex 容器，我们把一个容器的 display 属性值改为 flex 或者 inline-flex。完成这一步之后，容器中的直系子元素就会变为 flex 元素。

由于所有 CSS 属性都会有一个初始值，所以 flex 容器中的所有 flex 元素都会有下列行为：
元素排列为一行（flex-direction 属性的初始值是 row）。
元素从主轴的起始线开始。
元素不会在主维度方向拉伸，但是可以缩小。
元素被拉伸来填充交叉轴大小。

这会让你的元素呈线形排列，并且把自己的大小作为主轴上的大小。如果有太多元素超出容器，它们会溢出而不会换行。如果一些元素比其他元素高，那么元素会沿交叉轴被拉伸来填满它的大小。
拉伸填充交叉轴大小的前提是没有单独为该元素写样式（优先级问题）
