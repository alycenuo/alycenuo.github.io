前端学习-html篇

Created2024-04-16|Updated2024-05-05|知识笔记

番外

  • vscode的ctrl+B可以加粗，或者打开/关闭侧边栏。
  • shift+tab 左缩进
  • vscode中t添加注释快捷键为 ctrl+/
  • shift+alt+. 快速修复markdown语法错误
  • vscode有标签补全功能，例如想打<h1></h1>只需要打 h1然后回车补全即可
  • vscode ./ 可以提示当前路径的文件，可在插入图片时使用，提高效率，可参考 10、路径的内容
  • vscode写HTML文件时，可以直接无选择复制然后粘贴，删除更改或添加，提高效率
  • vscode写input的type的password没有提示可以生成<type="password">然后删除"password"再打""和p就可以有提示了。

正篇

1、在vscode中快速生成网页基本骨架

  • 英语状态下的 ! 号然后 回车

2、标签含义

  • html:整个网页
  • head：网页头部，存放给浏览器看的代码，例如css
  • body：网页主体，存放给用户看的代码，例如文字、图片
  • title：网页标题

3、标签关系

  • 嵌套关系-表现为包含
  • 并列关系-表现为对齐

4、注释的增加和删除

  • vscode中快捷键都为 ctrl+/
  • 写法为<!-- -->

5、标题标签

  • <h1></h1> 标题标签，一共6个级别h1~h6
  • h1 一个网页一般只用一次，用于标题和logo

6、段落标签

  • <p></p> 段落标签
  • 独立一行，行间有空隙，主要用于段落、描述等等。

7、换行和水平线标签

  • <br> 换行
  • <hr> 水平线

8、文本格式化标签

类型一（常用）

  • strong 加粗
  • em 倾斜
  • ins 下划线
  • del 删除线

strong的加粗对搜索引擎的权重有影响，对于重要内容，即使不需要加粗的效果，也可以用该标签，然后通过font-size取消加粗效果

类型二（了解）

  • b 加粗
  • i 倾斜
  • u 下划线
  • s 删除线

9、图片标签

  1. 用于插入图片
  2. 属性写法 属性名=”属性值”，不分顺序
  3. src 图片地址
  4. alt 替换文本 图片无法显示时显示的文字
  5. title 提示文本 鼠标悬停在图片上面时显示的文字
  6. width 图片的宽度 值为数字，没有单位，等比缩放
  7. heitht 图片高度，值为数字，没有单位，等比缩放
  8. width和height一般在css中设置，了解即可

10、路径

  1. 路径： 指的是查找文件时从起点到终点经历的路线
  2. 相对路径： 从当前文件位置出发查找目标文件
  3. 绝对路径： 从盘符出发查找目标文件，盘符根据计算机的不同指代不同
  • 相对路径
  1. . 代表当前文件所在文件夹
  2. .. 代表文件上一级文件夹
  3. / 进入某个文件夹里面
  • 绝对路径
  1. 包括文件的在线网址，比如图片网络链接
  2. 一般用于友情链接

11、超链接标签（页面跳转）

  1. 作用是跳转到其它页面
  2. href 指定跳转地址，可以是网址也可以是相对路径里的文件（猜测可用于网页内不同页面跳转）
  3. 添加target="_blank"属性可以在新窗口打开跳转页面而不关闭源网页
  4. 开发初期，不知道href的属性值可以用#号占位，表示空连接，不会跳转，后期知道具体地址再替换

11、音视频标签

  • 音频标签
  1. src代表音频地址，支持格式mp3、wav、ogg
  2. controls代表显示控制面板，loop代表循环播放，autoplay代表自动播放
  3. 当属性值和属性名相同，可以简写为一个，controls实际是是controls="controls"的简写，loop和autoplay同理
  4. 大部分浏览器都是禁止自动播放的
  • 视频标签
  1. 支持格式，mp4、webM、ogg
  2. loop、controls、autoplay和音频标签同理，muted代表静音播放
  3. 只有静音状态下自动播放才生效

12、列表

  • 有序列表
  1. 作用： 不需要规定顺序的区域
  2. ul嵌套li，ul是无序列表，li是条目
  3. ul标签里面只能包裹li标签，但li标签里面可以包裹任何内容
  • 有序列表
  1. 作用： 需要规定顺序的区域
  2. 其它与无序列表相同
  • 定义列表
  1. 主要用于网页底部友情链接、帮助等栏目
  2. dl里面只能包含dt和dd，dt和dd里面可以包含任何内容

13、表格和表格结构标签

  • 表格标签
  1. table嵌套tr，tr嵌套td/th
  2. table代表表格，tr代表行，th表头单元格，td内容单元格
  3. 默认没有线框，border="1"代表有线框，且线的像素大小为1
  • 表格结构标签
  1. 仅对浏览器而言，作用是让表格结构更加清晰
  2. thead 头部；tbody 表格主体；tfoot 表格底部
  3. 了解即可

14、合并单元格

  1. 跨行合并，保留最上的单元格，添加属性rowspan，属性值代表要合并的单元格数
  2. 跨列合并，保留最左单元格，添加属性colspan，属性值代表要合并的单元格数
  3. 合并后，其余单元格删除掉，或者改为注释
  4. 单元格不能跨结构合并

15、表单 input标签

  • 作用： 收集用户信息，例如注册、登录、搜索区域
  • placeholder input标签的占位文本（提示作用）
  • type不同的属性值会有不同的效果
  1. text 文本框，用于输入单行文本
  2. password 密码框
  3. radio 单选框
  4. checkbox 多选框
  5. file 上传文件
  • radio单选框
  1. name 属性名，控制分组，同组只能选一个（单选功能），属性值自定义
  2. checked 属性名，设置默认选项，单选和多选都可用，属性名与属性值相同
  • multiple 属性名和属性值相同，实现file上传文件的多选功能

16、表单 下拉菜单

  • select嵌套option
  • selected 默认选择
  • name、id、value用于发送数据，目前暂时忽视

17、文本域

  • cols和rows用于设置文本域大小，实际工作用css设置大小，一般不写
  • 文本框大小可以容易拖拽，实际工作为了不影响网页界面排布，一般禁用

18、label标签

  • 作用： 增大点击范围，提升用户体验
  • 支持label标签增大点击范围的表单控件有： 文本框、密码框、上传文件。单选框、多选框、下拉菜单、文本域等等。

19、按钮标签 button

  • submit 提交按钮，默认功能，即省略type时的功能
  • reset 重置按钮
  • button 普通按钮，默认没有功能，配合JavaScript使用可以实现任何功能
  • button功能需要指定区域才能使用，form标签可划定表单区域
  • action发送数据的地址，暂时忽视

20、无意义布局标签 div和span

  • 作用： 布局网页（划分网页区域，摆放内容）
  • div： 独占一行
  • span： 不换行
  • div被称为大盒子，span被称为小盒子
  • div可以通过height、width设置高宽，background设置背景颜色

21、常用字符实体

  • 作用： 在网页中显示预留字符

  • 空格的字符实体为： &nbsp;

  • 小于号<的字符实体为&lt;

  • 大于号>的字符实体为&gt;
