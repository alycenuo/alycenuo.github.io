正篇

1、css的定义和基本体验

  • 定义： 层叠样式表（全称Cascading Style Sheets），是一种样式表语言，用来美化HTML的内容。
  • 书写位置（内部样式表）： head标签内部，title标签下方添加style双标签，style标签里面书写css代码。
  • css的基础写法为选择器 { 属性名：属性值;}
  1. 所示代码的选择器为p，选择了body部分p标签的内容
  2. color和font-size是属性名，red和30px是属性值，后面注意跟 ;
  • 属性名和属性值成对出现可以称为键值对

2、css的引入方式

  • 内部样式表一般用于学习，css代码写在style标签里面，具体写法见1、css的定义和基本体验
  • 外部样式表主要用于开发使用
  1. css代码写在单独的.css文件中
  2. 在HTML使用link标签引入，link标签写在title下面，引入.css文件
  3. rel表示关系，stylesheet表示样式表

 

  • .css文件里面的写法与style标签内的写法一致
  • 行内样式： 配合JavaScript使用
  • css写在标签的style属性值里面

3、选择器-标签和类选择器

  • 选择器作用： 查找标签，设置样式，只有指定标签才能设置样式

 

  • 标签选择器： 使用标签名作为选择器，设置相同样式，无法差异化

 

  • 类选择器： 查找标签，差异化设置标签的显示效果
  1. 首先要自定义选择器，写法为.类名，类名一般用英语，且应当通俗易懂，可用-连接单词
  2. 使用类选择器，在标签添加class="类名"
  3. 一个类选择器可以给多个标签使用，但一个标签不允许加多个class属性，但是一个class属性可以有多个值

4、选择器-id和通配符

  • id选择器： 查找标签，差异化设置标签的显示效果
  • id选择器与类选择器相似，但是并不常用于设置css样式，一般配合JavaScript使用
  • 同一个id选择器在一个页面只能使用一次
  • 通配符选择器： 查找页面所有标签，设置相同样式，无需调用
  • 一般在工作初期使用，清除标签的默认属性，例如字体间距、颜色等默认属性，再进行设置

5、字体修饰属性

  • 浏览器右键点检查可打开调试工具，查看原网页的样式设置
  • 谷歌浏览器字体大小默认16px

属性

  • font-size 字体大小，必须带单位，不然不生效

  • fort-weight 字体粗细，有两种写法，一种是关键字normal表示不加粗，bold表示加粗，另一种是直接写数字例如700（加粗）、400（不加粗）等

  • font-style 字体倾斜，实际常用于清楚倾斜效果，例如清除em标签的倾斜效果，属性值有normal（不倾斜）italic（倾斜）

  • 行高，设置文本间距，行高的组成由上间距+字体高度+下间距组成

  • 测量行高的方法： 从一行文字的最顶到下一行文字的最顶

  • 利用行高垂直居中： 将行高属性值设置为跟盒子（div）的高度一致，只适用于不换行的情况

  • font-family 字体族,设置字体
  • 从左到右依次查找所写字体，找到哪个用哪个，最后那个是字体族名，网页开发一般使用无衬线字体
  • 可以到其它网页复制该代码
  • font 复合属性，节省代码量，用于设置网页的公共样式，再根据需要进行差异化
  • 属性值间用空格隔开
  • 必须按顺序书写，顺序为： 是否倾斜 是否加粗 字号/行高 字体
  • 字号、字体必须写，否则font属性不生效
  • 可以到其它网页复制该代码
  • text-indent 文本缩进
  • text-align 文本对齐，调整的是内容的对齐，而不改变标签的位置
  • 如果div里面是图片，那么会实现图片对齐
  • text-decoration 修饰线
  • color 颜色，有四种表示方法，分别是颜色关键字、rgb表示法、rgba表示法、十六机制表示法

6、调试工具

  • 打开方式： 菜单-检查，或者F12快捷键
  • 黄色叹号表示有问题，并且会指出出错的文件和出错的代码行数
  • 不勾选表示不使用该样式，并不改变本地文件，仅供调试

7、复合选择器

  • 后代选择器 选中某元素的后代元素
  • 子代选择器
  • 并集选择器 选中多组标签设置相同样式
  • 交集选择器 选择同时满足多个条件的元素
  • 伪类选择器 伪类表示元素状态，选中元素的某个状态为元素设置样式（例如鼠标悬停这种状态）
  • 超链接的四种状态 分别是 访问前、访问后、鼠标悬停、点击时
  • 如果要给超链接设置以上四个状态，需要按LVHA的顺序书写
  • 实际工作中并不常写四种状态，而是通过a直接给四种状态设置同样样式，然后写一下:hover差异化

8、css特性

  • 继承性： 子集默认继承父级的文字控制属性
  • 如果原标签有样式，则优先使用自带的样式，包括默认样式，例如a标签有默认颜色，h标签有默认大小，所以不会继承父级的设置
  • 实际工作中，可以把文字属性写在body标签中，实现整体风格的统一，节省代码量
  • 层叠性
  1. 相同属性会覆盖，后面的css属性覆盖前面的css属性
  2. 不同的属性会叠加，不同的css属性都生效
  • 优先级 即权重，当一个标签使用了多种选择器时，基于不同种类的选择器的匹配规则
  • 规则 选择优先级高的样式生效
  • 公式 通配符选择器<标签选择器<类选择器<id选择器<行内样式<！important（总的来说，选中标签的范围越大，优先级越低）
  • !important 提高权重到优先级最高，慎用
  • 优先级 叠加计算规则
  • 如果是复合选择器，则需要权重叠加计算
  • (行内样式，id选择器个数，类选择器个数，标签选择器个数)

规则：

  • 从左到右依次比较个数，同一级个数多的优先级高，如果个数相同，则向后比较
  • ！important权重最高
  • 继承权重最低

10、背景属性

  • background-image 背景图
  • 浏览器的背景图片默认平铺效果，如果图片过小会复制图片铺满盒子
  • background-repeat 背景图平铺方式
  • background-position 属性名，控制背景图位置
  • 属性值 水平方向位置 垂直方向位置
  1. 有两种写法，一种是关键字写法，一种是坐标写法
  2. 两种写法可以混用
  3. 关键字写法可以颠倒取值顺序
  4. 关键字有left、right、center、top、bottom
  5. 坐标写法单位为px，数字可正可负

特殊写法

  • 可以只写一个关键字，另一个方向默认居中；
  • 数字只写一个值表示水平方向，垂直方向居中
  • background-size 背景图缩放
  • background-attachment 背景图固定，让背景不随着元素内容滚动
  • background 背景复合属性
  • 属性值 背景色 背景图 背景图平铺方式 背景图位置/背景图缩放 背景图固定
  • 不区分顺序

11、显示模式

块级元素素（例如div）

  1. 独占一行
  2. 宽度默认是父级的100%
  3. 添加宽高属性生效

行内元素（例如span）

  1. 一行可显示多个
  2. 设置宽高不生效
  3. 宽高尺寸由内容撑开

行内块元素（例如图片）

  1. 一行可以显示多个
  2. 设置宽高生效
  3. 宽高尺寸也可以由内容撑开

转换显示模式

  • 属性名： display
  • 属性值 block（块级）、inline-block（行内块）、inline（行内）
  • inline不常用
