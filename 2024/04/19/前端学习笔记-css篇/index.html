正篇

1、css的定义和基本体验

  • 定义： 层叠样式表（全称Cascading Style Sheets），是一种样式表语言，用来美化HTML的内容。
  • 书写位置（内部样式表）： head标签内部，title标签下方添加style双标签，style标签里面书写css代码。
  • css的基础写法为选择器 { 属性名：属性值;}
  1. 所示代码的选择器为p，选择了body部分p标签的内容
  2. color和font-size是属性名，red和30px是属性值，后面注意跟 ;
  • 属性名和属性值成对出现可以称为键值对

2、css的引入方式

  • 内部样式表一般用于学习，css代码写在style标签里面，具体写法见1、css的定义和基本体验
  • 外部样式表主要用于开发使用
  1. css代码写在单独的.css文件中
  2. 在HTML使用link标签引入，link标签写在title下面，引入.css文件
  3. rel表示关系，stylesheet表示样式表

 

  • .css文件里面的写法与style标签内的写法一致
  • 行内样式： 配合JavaScript使用
  • css写在标签的style属性值里面

3、选择器-标签和类选择器

  • 选择器作用： 查找标签，设置样式，只有指定标签才能设置样式

 

  • 标签选择器： 使用标签名作为选择器，设置相同样式，无法差异化

 

  • 类选择器： 查找标签，差异化设置标签的显示效果
  1. 首先要自定义选择器，写法为.类名，类名一般用英语，且应当通俗易懂，可用-连接单词
  2. 使用类选择器，在标签添加class="类名"
  3. 一个类选择器可以给多个标签使用，但一个标签不允许加多个class属性，但是一个class属性可以有多个值

4、选择器-id和通配符

  • id选择器： 查找标签，差异化设置标签的显示效果
  • id选择器与类选择器相似，但是并不常用于设置css样式，一般配合JavaScript使用
  • 同一个id选择器在一个页面只能使用一次
  • 通配符选择器： 查找页面所有标签，设置相同样式，无需调用
  • 一般在工作初期使用，清除标签的默认属性，例如字体间距、颜色等默认属性，再进行设置

5、字体修饰属性

  • 浏览器右键点检查可打开调试工具，查看原网页的样式设置
  • 谷歌浏览器字体大小默认16px

属性

  • font-size 字体大小，必须带单位，不然不生效

  • fort-weight 字体粗细，有两种写法，一种是关键字normal表示不加粗，bold表示加粗，另一种是直接写数字例如700（加粗）、400（不加粗）等

  • font-style 字体倾斜，实际常用于清楚倾斜效果，例如清除em标签的倾斜效果，属性值有normal（不倾斜）italic（倾斜）

  • 行高，设置文本间距，行高的组成由上间距+字体高度+下间距组成

  • 测量行高的方法： 从一行文字的最顶到下一行文字的最顶

  • 利用行高垂直居中： 将行高属性值设置为跟盒子（div）的高度一致，只适用于不换行的情况

  • font-family 字体族,设置字体
  • 从左到右依次查找所写字体，找到哪个用哪个，最后那个是字体族名，网页开发一般使用无衬线字体
  • 可以到其它网页复制该代码
  • font 复合属性，节省代码量，用于设置网页的公共样式，再根据需要进行差异化
  • 属性值间用空格隔开
  • 必须按顺序书写，顺序为： 是否倾斜 是否加粗 字号/行高 字体
  • 字号、字体必须写，否则font属性不生效
  • 可以到其它网页复制该代码
  • text-indent 文本缩进
  • text-align 文本对齐，调整的是内容的对齐，而不改变标签的位置
  • 如果div里面是图片，那么会实现图片对齐
  • text-decoration 修饰线
  • color 颜色，有四种表示方法，分别是颜色关键字、rgb表示法、rgba表示法、十六进制表示法

十六进制
十六进制（简写为hex或下标16）是一种基数为16的计数系统，是一种逢16进1的进位制。通常用数字0、1、2、3、4、5、6、7、8、9和字母A、B、C、D、E、F（a、b、c、d、e、f）表示，其中:A~F表示10~15，这些称作十六进制数字

6、调试工具

  • 打开方式： 菜单-检查，或者F12快捷键
  • 黄色叹号表示有问题，并且会指出出错的文件和出错的代码行数

7、复合选择器

  • 后代选择器 选中某元素的后代元素
  • 子代选择器
  • 并集选择器 选中多组标签设置相同样式
  • 交集选择器 选择同时满足多个条件的元素
  • 伪类选择器 伪类表示元素状态，选中元素的某个状态为元素设置样式（例如鼠标悬停这种状态）
  • 超链接的四种状态 分别是 访问前、访问后、鼠标悬停、点击时
  • 如果要给超链接设置以上四个状态，需要按LVHA的顺序书写
  • 实际工作中并不常写四种状态，而是通过a直接给四种状态设置同样样式，然后写一下:hover差异化

8、css特性

  • 继承性： 子集默认继承父级的文字控制属性
  • 如果原标签有样式，则优先使用自带的样式，包括默认样式，例如a标签有默认颜色，h标签有默认大小，所以不会继承父级的设置
  • 实际工作中，可以把文字属性写在body标签中，实现整体风格的统一，节省代码量
  • 层叠性
  1. 相同属性会覆盖，后面的css属性覆盖前面的css属性
  2. 不同的属性会叠加，不同的css属性都生效
  • 优先级 即权重，当一个标签使用了多种选择器时，基于不同种类的选择器的匹配规则
  • 规则 选择优先级高的样式生效
  • 公式 通配符选择器<标签选择器<类选择器<id选择器<行内样式<！important（总的来说，选中标签的范围越大，优先级越低）
  • !important 提高权重到优先级最高，慎用
  • 优先级 叠加计算规则
  • 如果是复合选择器，则需要权重叠加计算
  • (行内样式，id选择器个数，类选择器个数，标签选择器个数)

规则：

  • 从左到右依次比较个数，同一级个数多的优先级高，如果个数相同，则向后比较
  • ！important权重最高
  • 继承权重最低

10、背景属性

  • background-image 背景图
  • 浏览器的背景图片默认平铺效果，如果图片过小会复制图片铺满盒子
  • background-repeat 背景图平铺方式
  • background-position 属性名，控制背景图位置
  • 属性值 水平方向位置 垂直方向位置
  1. 有两种写法，一种是关键字写法，一种是坐标写法
  2. 两种写法可以混用
  3. 关键字写法可以颠倒取值顺序
  4. 关键字有left、right、center、top、bottom
  5. 坐标写法单位为px，数字可正可负

特殊写法

  • 可以只写一个关键字，另一个方向默认居中；
  • 数字只写一个值表示水平方向，垂直方向居中
  • background-size 背景图缩放
  • background-attachment 背景图固定，让背景不随着元素内容滚动
  • background 背景复合属性
  • 属性值 背景色 背景图 背景图平铺方式 背景图位置/背景图缩放 背景图固定
  • 不区分顺序

11、显示模式

块级元素素（例如div）

  1. 独占一行
  2. 宽度默认是父级的100%
  3. 添加宽高属性生效

行内元素（例如span）

  1. 一行可显示多个
  2. 设置宽高不生效
  3. 宽高尺寸由内容撑开

行内块元素（例如图片）

  1. 一行可以显示多个
  2. 设置宽高生效
  3. 宽高尺寸也可以由内容撑开

转换显示模式

  • 属性名： display
  • 属性值 block（块级）、inline-block（行内块）、inline（行内）
  • inline不常用

15、盒子模型

盒子模型主要组成部分

  • 内容区域 width&height
  • 内边距 padding（出现在内容与盒子边缘之间）
  • 边框线 border
  • 外边距 margin（出现在盒子外面，拉开盒子间距离）

边框线（border）

属性名： border（bd）
属性值： 边框线粗细 线条样式 颜色（不区分顺序）

常用线条样式
| 属性值 | 线条样式|
|—-|—-|
| solid | 实线 |
|dashed | 虚线 |
| dotted | 点线 |

设置单方向的边框线
属性名： border-方位名词（例如bdl代表左边框线）
方位名词： top、right、bottom、left

内边距（padding）与多值写法

  • 与border属性一样，每个方向可单独设置。

  • 多值写法 可以简化代码

取值个数 含义
一个 四个方向接为该值
四个 上、右、下、左
三个 上、左右、下
两个 上下、左右

记忆方法： 顺时针旋转，依次分配，未被分配取对位方向相同值

盒子尺寸计算

默认情况
盒子尺寸 = 内容尺寸 + border尺寸 + 内边距尺寸

解决办法

  • 手动做减法，减掉border/padding的尺寸
  • 内减模式： 添加属性 box-sizing:border-box让浏览器自动调整大小

外边距（margin）与版心居中

与内边距相似，外边距也可以给四个方向单独设置外边距，并且也可以使用多值写法

版心居中： 让左右外边距相同即可，一般设置为auto让浏览器自动计算，版心居中效果要求盒子必须有宽度

18、外边距问题

合并现象

场景： 垂直的并列元素，上下margin会合并，只取两个margin中的较大值生效

  • 由此，可以只设置外边距较大的盒子，节省代码量

塌陷问题（坑爹现象）

场景： 父子级的标签，子级添加上外边距会产生塌陷问题，导致父级一起向下移动

解决方法：

  • 取消子级margin，父级设置padding（推荐）
  • 父级设置 overflow:hidden
  • 父级设置 bored-top

后两种解决方法实际上是让浏览器找到了正确的边缘

19、行内元素-内外边距问题

场景： 行内元素添加margin和padding，无法改变元素垂直位置

  • 解决方法： 给行内元素添加 line-height 可以改变垂直位置

20、盒子模型-圆角和盒子阴影

圆角

作用： 设置元素的外边框为圆角
属性名： border-radius
属性值： 数字+px/百分比（属性值是圆角半径）

常见应用

  • 正圆形状 可以作头像，给正方形盒子设置圆角属性值为宽高的一半/50%
  • 胶囊形状 给长方形盒子设置圆角属性为盒子高度的一半

阴影

作用： 给元素设置阴影效果
属性名： box-shadow
属性值： X轴偏移量 Y轴偏移量 模糊半径 扩散半径 颜色 内外阴影（不写默认外阴影）

注意：

  • X轴偏移量和Y轴偏移量 必须书写
  • 默认是外阴影，内阴影需要添加 inset

21、标准流的概念

标准流： 也叫文档流，指的是标签在页面中的默认排布规则，例如块级元素独占一行。

22、浮动流（了解）

作用： 让块元素水平排列，实际可以用于图文混排，随着技术进步不再常用，更加常用flex

属性名： float

属性值:

  • left： 左对齐
  • right： 右对其

特点：

  • 浮动后盒子顶对其
  • 浮动后盒子具备行内块特点
  • 脱标现象： 添加了浮动属性会脱离的标准流，不再占用标准流的位置，在浏览器眼里相当于不存在该内容。因此，同一行的元素，要么都加标准流，要么不加。

23、清除浮动的方法

场景： 浮动元素会脱标，如果父级没有高度，子级无法撑开父级高度，可能导致页面布局错乱

  • 因为脱标的盒子作为内容无法撑开盒子的高

方法一： 额外标签法

  • 在父元素内容的最后添加一个块级元素，设置CSS属性 clear:both

方法二： 单伪元素法

双伪元素法（推荐）

  • 双伪元素还可以解决塌陷问题

方法四： overflow

  • 父级元素添加CSS属性 overfloe:hidden
  • 该属性让浏览器能正确检查父级的范围

24、Flex

认识Flex

Flex布局也叫弹性布局，不会产生浮动布局中的脱标现象，布局网页更加简单、灵活

设置方式： 给父级设置 display:flex，子元素可以自动挤压或拉伸

组成部分：

  • 弹性容器
  • 弹性盒子
  • 主轴： 默认在水平方向
  • 侧轴： 默认在垂直方向

主轴对齐方式

属性名： justify-content

属性值 效果
flex-start 默认值，从起点开始依次排列
flex-end 弹性盒子从终点开始排列
center 沿主轴居中排列
space-between 沿主轴均匀排布，空白间距均分在弹性盒子之间
space-around 沿主轴均匀排布，空白间距均分在弹性盒子两侧
space-evenly 沿主轴均匀排布，弹性盒子与容器之间间距相等

侧轴对齐方式

属性名

  • align-items 当前弹性容器内所有弹性盒子的侧轴对齐方式（给弹性容器设置）
  • align-self 单独控制某个弹性盒子的侧轴对齐方式（给弹性盒子设置）
属性名 效果
stretch 弹性盒子沿侧轴被拉伸至铺满容器（没有设置侧轴方向尺寸才默认可拉伸）
center 居中
flex-start 起点开始依次排列
flex-end 终点开始依次排列

修改主轴方向

属性名： flex-direction
属性值： column， 将主轴修改为垂直方向，从上到下

  • 有其它属性值不常用，暂忽略
  • 修改主轴方向，侧轴方向自动变换到对应方向

弹性伸缩比

作用： 控制弹性盒子的主轴方向的尺寸
属性名： flex
属性值： 整数数字，表示占用父级剩余尺寸的份数

弹性盒子自动换行

弹性盒子可以自动挤压或拉伸，默认情况下，所有弹性盒子都在一行显示
属性名： flex-wrap

属性值

  • warp： 换行
  • nowrap： 不换行（默认）

行对齐方式

属性名： align-content
属性值： 与主轴对齐方式相同

属性值 效果
flex-start 默认值，从起点开始依次排列
flex-end 弹性盒子从终点开始排列
center 沿主轴居中排列
space-between 沿主轴均匀排布，空白间距均分在弹性盒子之间
space-around 沿主轴均匀排布，空白间距均分在弹性盒子两侧
space-evenly 沿主轴均匀排布，弹性盒子与容器之间间距相等
  • 不同的是，行对齐方式对单行不生效

25、定位

定位： 灵活改变盒子在网页中的位置，可让盒子重叠

实现

  1. 定位模式： position
  2. 边偏移
  • left
  • right
  • top
  • bottom

相对定位

一般不单独使用，而是配合其它定位一起使用

绝对定位

position: absolute
使用场景： 子级绝对定位，父级相对定位（子绝父相）

定位居中

实现步骤

  1. 绝对定位
  2. 水平、垂直边偏移为50%
  3. 子级向左、上移动自身尺寸的一半
  • 实现方法一： 左、上的外边距为尺寸的一半
  • 实现方法二： transform: translate(-50%,-50%)

固定定位

元素的位置不随网页滚动而改变

26、堆叠层级 z-index

默认效果： 按照标签书写顺序，后来者居上

作用： 设置定位元素的层级顺序，改变定位元素的显示顺序

27、CSS精灵

CSS精灵，也叫CSS Sprite，是一种网页图片应用处理方式。把网页一些背景图片整合到一张图片文件中，再通过background-position精确定位背景图片的位置

优点： 减少服务器被请求次数，减轻服务器的压力，提高页面加载速度

实现步骤：

  1. 创建盒子，盒子尺寸与小图尺寸相同
  2. 设置盒子背景图为精灵图
  3. 添加background-position属性，改变背景图位置
  • 使用PxCook测量小图片左上角坐标
  • 取负数坐标为background-position属性值（向左上移动图片位置）

29、修饰属性-垂直对其方式（vertical-align）

浏览器默认将行内、行内块当作文字按照基线对齐，导致图片和文字不齐整，例如图片下面有空白

属性名： vertical-align

属性值 效果
baseline 基线对齐（默认）
top 顶部对齐
middle 居中对齐
bottom 底部对齐

30、过渡属性transition

作用： 可以为一个元素在不同状态间切换的时候添加过渡效果
属性名： transition（复合属性）
属性值： 过渡的属性 花费的时间

  • 过渡的属性可以是具体的CSS属性
  • 也可以为all（两个状态属性值不同的所有属性，都产生过渡效果）
  • transition设置给元素本身

30、透明度opacity

作用： 设置整个元素的透明度（包含背景和内容），rgba只能设置背景透明度
属性名： opacity
属性值： 0-1

  • 0： 完全透明
  • 1： 不透明

31、光标类型cursor

作用： 鼠标悬停在元素上是指针显示样式
属性名： cursor

属性值 效果
default 默认值，通常是箭头
pointer 小手效果，提示可以点击
text 工字型，提示可以输入
move 十字型，提示可以移动
