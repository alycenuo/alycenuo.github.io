《吞食魔果的人-毛姆》

Created2024-05-17|Updated2024-05-24

情节概括

倒金字塔式概括：
托马斯·威尔逊快乐生活25年后，畏惧自杀，苟活六年死去。
（核心事件）

35岁的托马斯·威尔逊辞去银行经理的工作，用所有财产买了一份25年期的保险，决定25年间不工作尽情享受生活，保险到期自杀。但他的意志在惬意的生活中消磨殆尽，保险到期后拖欠房租、赊账，野人般生活六年后最终死在一个小山坡。
（该段内容基本是核心事件的展开或者具体化）

（次要情节，例如托马斯·威尔逊的生活、台词、行为、他人的评价等等）

→ 总的来说，倒金字塔第一段要突出并且概括核心事件，后续的一或两段是第一段的展开或者仅仅相对不够核心但仍然重要的内容。跟核心句是总分关系或者包含关系。而后续补充性的内容可以认为是并列关系。

四幕式概括（起承转合）：
托马斯·威尔逊的太太四年前死于支气管肺炎后，女儿去跟外婆，不久患上血中毒去世，没有兄弟姐妹，没有需要赡养的父母儿女，35岁便孑然一身，托马斯·威尔逊思考起人生意义。

最终认定自由、诗意、随心所欲才是生活，毅然辞去银行经理的工作，用所有财产买下一份25年期的年金保险，以便不用工作也可以维持还算惬意的生活，尽情享受生命，等保险到期，再了无遗憾自杀。
25年期间，托马斯·威尔逊在卡普里游泳、爬山、读书，欣赏美景，尽情人生。

然而25年过去，托马斯·威尔逊意志变得软弱无力，不断推后自我了断的日期，却又没有资金来源，长期拖欠房租，当地商家也不再愿意赊账，走投无路，烧炭自杀——但不知是意外，还是决心有所动摇，房屋通风良好，托马斯·威尔逊并没有自杀成功，最后被好心的房东太太收容。
房东太太将他安置在小鹏，给他几条毯子，粗茶淡饭，没有肉食，礼拜天只给顿通心面，她老公对他苛刻，使役他提水、清扫牛棚做杂活。

托马斯·威尔逊平日在山里到处乱跑，见到外人慌不择路，撒腿就跑。潦倒生活六年后，他的尸体某早晨在山坡被发现。

→ 段落间主要是递进关系，不太自由，想要插入一些补充性内容很影响结构，但故事结构上较为完整。

主观分析

第一部分 初识 角色入场

以第一视角议论入文：世间之人对当前境遇，要么愤愤不平，要么安之若素，随之任之，极少人能掌握自我命运 → 对比之下，托马斯·威尔逊就显得奇特，但并不详说为什么奇特，戛然而止，设置悬念 → 正式开始剧情 → 卡普里岛、与托马斯·威尔逊初识、大体介绍一下地理环境、描写托马斯·威尔逊外貌、与托马斯·威尔逊同行，通过移步换景的方式进行场景切换介绍当地社会环境、以对话过渡 → 最终得出一个结论：在我看来，他绝无超乎寻常之处，一个生意人罢了，靠着金边优质证券有项不错的退休收入。我看你的故事只不过是卡普里一般的街谈巷议而已。

第二部分 海滩相遇+概括叙事 过渡

进一步塑造托马斯·威尔逊，塑造方式主要有描写和对话，描写沙滩情景下托马斯·威尔逊的外在，交谈写他读书，以及生活方式。但是仍然没有显示其足够独特之处。
通过叙事的方式，“我又数次邂逅威尔逊”，然后概括地对托马斯·威尔逊进行一个整体刻画。

第三部分 夜游对饮 揭示悬念

两人独处交谈时，托马斯·威尔逊透露了自己的过去，经历丧妻丧女的人生巨变后，思考人生意义，辞去银行经理的职位，卖一份保险，在25年内不工作，尽情玩乐。

夜游结束，“我”被邀请到托马斯·威尔逊的家里，描写其住所，通过想象的方式去描写其日常生活，间夹评价。
最后以“我”离开卡普里岛结尾 → 快进时间。

第四部分 揭示结局

十三年后，“我”重返卡普里岛 → 通过与朋友交流介绍托马斯·威尔逊的结局 → 长期的安逸生活，消磨了托马斯·威尔逊，保险到期失去资金来源后，畏惧自我了断，又不愿过，最后被当成傻子由好心的房东收养。生活潦倒没有自尊：

阿孙塔来医院领他回去，他一声不吭跟着走了。他似乎丧失了自主意志。房东太太收容他迄今已有两年。

“你知道，舒适是不用谈了，”朋友说，“他们草草给他搭了张东倒西歪的床铺，给了他几条毯子。小棚没有窗户，冬天滴水成冰，夏天就像火炉。吃的是粗茶淡饭。农家伙食你知道，礼拜天吃顿通心面，难得一尝肉味。”

“那他怎么打发时间呢？”

“他在山里到处乱跑。有两三次，我想见见他，可是没辙。他一见有人来，像只野兔撒腿便跑。阿孙塔有时下山来跟我闲聊，我会给她一点钱，让她替他买回烟丝，可谁知道他最终拿到烟丝没有。”

“他们待他还好吗？”我问。

“阿孙塔是好心肠，这个我有把握。她把他当做小孩。恐怕她老公就没有这份善心了，老是抱怨收留这么个人花销有多大。我并不认为房东生性残忍，或有诸如此类的弱点，可我看房东对他有些苛刻，要他提水，清扫牛棚，做这样那样的杂活。”

跟朋友散步时，托马斯·威尔逊在树丛后窥视“我们”，朋友让“我”可以不要注意托马斯·威尔逊，因为——

“瞧，威尔逊，”朋友突然说，“别看他，那样你只会吓着他。一直往前走。”

双目低垂着看路，我自顾自往前走，可眼角的余光扫到一个藏身在地中海橄榄树后的男子。我们走近时，他潜伏着一动不动，但我可感知他在紧盯着我们。待我们走过，我听见一阵噼噼啪啪狂奔的脚步声。像一头被追逐的猎物，威尔逊逃着找安全的藏匿处去了。那是我此生最后一次看到他。

主观评价

不得不说，第一、二部分真的无聊又冗长，大量长描写，而且文字并不流畅易懂（当然，极有可能是翻译的问题），不过无论如何，个人觉得这两个部分完全不需要那么长的篇幅……总给我一种凑字数的感觉。

而且，悬念其实不够明显，第一遍看的时候并没有留意到设置了悬念，就是说 → 不够突出重点，主次不够分明，冗余成分过多。

此外，这个悬念的分量不足，以致于后面揭示悬念的时候觉得“就这种东西你卖那么长一段关子”，不能满足预期，有点不满。

第三部分也是，展现方式不够主次分明，而且不够显得托马斯·威尔逊意气风发，以便与第四部分的落寞产生更加强烈的对比。

但是，第四部分很好，个人觉得第四部分才是小说的高潮部分。漂亮的地方在于情感的突变，这是通过对比实现的。曾经“决心坚定、勇于追求自我”的人，如今却意志消沉，毫无自尊。总之很触动人。

前面的所有部分都可以认为是第四部分的铺垫，那么冗长的铺垫（尽管不够有针对性，但从另一种角度看，更加真实自然，不显得为文造景），结果却是稍显克制、平常化的结尾，隐而不发，拉开了弓却又缓缓放松，有种情感不得抒发的压抑感。
蓄势却不爆发，而是平常化处理，也是一种技巧呢。毛姆写奇特的人和奇特的心真的很厉害……

全篇最大缺陷是“语言”，但是当时，这种写法是主流，当时的作家总是习惯对人物外貌、地理环境进行大段描写，哪怕压根没什么必要，对剧情没什么作用……不过，也可以认为是为了追求真实性故意写的冗余……但个人认为，短篇小说不必如此。

正如题目所说，托马斯·威尔逊是吞食魔果的人，这颗魔果名为安逸。魔果常常伪装成美好的事物，初尝恰似甜美，暗则侵蚀灵魂。正如谎言以真话铺垫才好骗人。

分析小说时，总感觉缺少术语，或者说缺乏适合的表达，分析的角度也很狭隘主观……明显感觉到自己知识不足，见识不足。
