1、 JavaScript介绍

是什么

  • 是一种运行在客户端（浏览器）的编程语言，实现人机交互效果

作用

  • 网页特效（监听用户的一些行为让网页作出对应的反馈）
  • 表单验证（判断表单数据合法性）
  • 数据交互（获取后台数据，渲染到前端）
  • 服务端编程（node.js）

JavaScript的组成

  • ECMAScript： 规定js基础语法，如变量、分支语句、循环语句、对象等等
  • Web Apis：
    • DOM 操作文档，比如对页面元素进行移动。添加删除等操作
    • BOM 操作浏览器，比如页面弹窗，检测窗口宽度，存储数据到浏览器等等

https://developer.mozilla.org/zh-CN/

2、JavaScript的书写位置

内部JavaScript

  • 直接写在html文件里面，用script标签包住
  • 规范： script标签写在</body>上面。浏览器加载代码是按照顺序从上到下加载HTML，如果先加载的JavaScript期望修改的文件在JavaScript之后，可能由于HTML未加载而失效

外部JavaScript

  1. script标签间无需写代码，否则会被忽略
  2. 外部引入会使代码更加有序，易于复用，html也会更加易读

内联JavaScript
代码写在标签内部，vue框架会用到此模式

3、JavaScript的输入输出语句、顺序、结束符

代码执行顺序
一般来说会从上到下执行，但有alert和prompt时会优先执行

结束符
JavaScript的结束符为分号;，但也可以不写，应当尽可能保持统一，要么都写，要么不写

4、变量

变量是存储数据的容器，它可以收集数据方便调用，让计算机有“记忆”，计算机通过调用变量名获取对应的数据；也可以储存未知的数据（用户不确定的输入）

变量的使用

声明变量（即定义、创建变量）

  • 语法： let 变量名

变量赋值

  • 例如： age = 18
  • 从右往左赋值

实际使用

  • 声明的同时赋值(又称变量的初始化)： let age = 18

更新变量

  • 变量赋值后可通过重新赋值来更新
  • 同一变量名不允许重复声明

同时声明多个变量

可读性不如一行声明一个变量，代码长度可以完成代码后进行打包缩短，若考虑可读性，不建议同时声明多个变量

收集输入并且输出（练习）

输入 → 内部处理 → 输出

交换变量数据（练习）

关键思路是利用中间变量

变量的本质、书写规则和规范

变量的本质是在内存里申请一个小地方储存数据，内存溢出的原因之一便是变量过多或变量储存的数据过大，内存不足

规则

  • 不能用关键字，如let、var、if等等
  • 只能用下划线、字母、数字、$，且不能以数字开头
  • 字母严格区分大小写，Age ≠ age

规范

  • 起名应该有意义
  • 小驼峰命名法，第一个单词首字母小写，后续单词首字母大写

关于var（了解）

在旧版本的JavaScript中使用var来声明变量

  • 可以先使用再声明（不合理）
  • var声明过的变量可以重复声明（不合理）
  • 变量提升、全局变量、没有块级作用域等

数组（array）

将一组数据储存在单个变量名下的方式

  • 数组是有序的，默认从0号开始，该序号又称索引号
  • 数组可以储存任意类型的数据
  • 元素： 数组中保存的每个数据都叫数组元素
  • 下标： 即索引号
  • 长度： 数组中数据的个数，可通过数组的length属性获得

其它
中文要加引号，否则可能会报错，原因之一是：中文可以作为变量名（怪东西），不加引号意味着将该中文视为变量，然后会因为未定义该变量而报错

常量

概念： 使用const声明的、不会改变的变量称为常量

  • 常量不允许重新赋值，声明的时候必须赋值（初始化）

例如： const pi = 3.1415

5、数据类型

js的数据类型主要分成两大类

  • 基本数据类型： number（数字型）、string（字符串型）、boolean（布尔型）、undefined（未定义型）、null（空类型 跟空对象全等）
  • 引用数据类型： object（对象）

js是弱数据类型的语言，变量属于哪种数据类型必须赋值后才能确定

number型

正数、负数、小数等统称为 数字类型

算术运算符

  • 加减乘除对应的运算符为： + - * /
  • 取模（取余数）的运算符为： %（常用于判断是否为整数）
  • 运算顺序： 加减乘除余，括号优先，平级从左到右

NaN

  • not a number，代表一个计算错误，例如字符串-2会得到一个NaN
  • NaN是粘性的，任何对NaN的的操作都会返回NaN

字符串型（string）

  • 通过单引号、双引号或反引号包裹的数据都叫字符串，单引号和双引号没有本质区别
  • 数字被单双引号包裹不再视为数字，而是字符串，不可进行数字运算
  • 无论单双引号都必须成对使用
  • 单双引号可以互相嵌套，但是不能自己嵌套自己（因为同种引号会找离自己最近的另一个引号成对）
  • 必要时可以通过转义字符\输出单双引号

关于+号
在数值运算中表示相加，但当前后有一方为字符串时表示拼接

模版字符串

  • 用于拼接字符串和变量
  • 语法： ${变量}

布尔型、未定义类型、空类型

布尔型（boolean）

  • 表示肯定或否定，只有两个固定的值true和false，用于判断，例如document.write(3>4)会返回一个false

未定义类型（undefined）

  • 只声明未定义的情况下，变量的默认类型为undefined
  • 实际工作： 声明一个变量，然后等待传输过来的数据，可以通过检测该变量是否为undefined来判断是否有数据传过来

空类型（null）

  • 表示无、值未知（该类型本身似乎是不太合理的）
  • 和undefined的区别： null表示赋值了但是内容为空，null+1会返回1；后者表示没有赋值，会返回NaN
  • 实际使用： 将来有个变量里面存放的是一个对象，但是对象还没有创建好，可以先给个null

检测数据类型

  1. typeof x
  2. typeof(x)

6、数据类型转换

使用表单、prompt获取的数据默认是 字符串类型

隐式转换

规则

  • +号两边只要有一个是字符串，都会把另外一个转换成字符串
  • 除了+号外，其余-*/等运算符会把数据转换成数字类型

显式转换

规则

  • Number(数据) 如果有非数字，结果为NaN
  • parseint(数据) 只保留整数
  • parseFloat(数据) 可以保留小数

parseFloat（abc12.5）会返回NaN，parseFloat（12.5abc）会返回12.5

在prompt前面直接加+也可以转换成数字型，如+prompt()

9、表达式和语句

表达式

  • 是可以被求值的代码，JavaScrip引擎会将其计算出一个结果，例如1+1

语句

  • 语句是一段可以执行的代码，不一定有值，比如prompt、if语句等等

区别
表达式可以求值，所以它可以写在赋值语句的右侧，而语句不一定有值，因此不是所有语句都能用于赋值

三大流程控制语句

  • 从上到下执行，叫顺序结构
  • 根据条件选择执行代码，叫分支结构
  • 某段代码被重复执行，叫循环结构

19、函数

函数
function： 设计为执行特定任务的代码块，可以把相同或相似逻辑的代码“封装”起来，通过函数调用执行这些被“封装”的代码逻辑，有利于精简代码和复用
前面的alert()、prompt()、和console.log()都是已经封装好的js函数

命名建议： 尽量用能体现其功能的动词或动+名短语，如getSum（求和）、loadArr（加载数组）

函数的使用和调用

函数传参、设置默认值

上面的函数是“死”的，不够灵活，因此引入参数这一概念

例子中的start和end交“形参”，是形式上的参数，而3和100叫“实参”，真正参与语句的参数
此外，为了严谨性，往往需要设置参数默认值，有实参时，默认值不生效

函数的返回值

语法：return 数据

  • return关键字能将内部的执行结果交给函数外部使用
  • return后面的代码不会被执行，会立即结束当前函数，return后面的数据不能换行写
  • 函数可以没有return，这种情况返回值为undefined
  • return只能返回一个值，若想返回多个值，可以用数组

函数的补充

  • 浏览器断点调试函数时，得按F11而不是F10才能进入函数内部（记得先刷新）
  • 如果函数名重复，后面的函数会覆盖前面的函数
  • 如果实参多于形参，多出来的实参不参与运算；如果形参少于实参，多出来的形参会自动填充undefined

20、作用域

一段代码中所用的名字并不总是有效和可用的，限定这个名字可用性的代码范围就是这个名字的作用域，作用域提高了程序逻辑的局部性，增强了程序的可靠性

全局作用域： 作用于代码执行的环境，整个script标签或单独的js文件
局部作用域： 例如函数内的代码环境

由此，根据作用域的不同可以分出全局变量和局部变量

  • 如果函数内部，变量没有声明，直接赋值，也当全局变量看，但强烈不推荐

变量的访问原则

  • 代码至少有一个作用域
  • 如果函数里面有函数，并且变量名相同，在能够访问到的情况下，先局部再全局

21、匿名函数

匿名函数，与具名函数相对，指没有函数名的函数，function (){}

一般来说，函数不调用无法执行，因此使用匿名函数有两种特殊的方法：函数表达式和立即执行

函数表达式

与具名函数的区别：具名函数可以在任意地方调用，而函数表达式必须先声明再调用

立即执行函数

避免全局变量之间的污染，比如使用第三方的代码，可能与本身自带的变量名产生冲突，但是用立即执行函数就不会再冲突

22、逻辑中断

短路：只存在于&&和||中，满足一定条件会让右边的代码不执行
原因：通过左边能得到整个式子的结果，因此没必要再判断右边
运算结果： 无论&&还是||，运算结果都是最后被执行的表达式值，一般用于变量赋值

基于逻辑中断，开发中，函数默认值有另一种写法

补充-隐式转换

  1. 字符串加法"" + 1结果为 “1”（字符串）
  2. 减法会使空字符串转化为0
  3. null经过数字转换变为0
  4. undefined数字转换变为NaN

24、知识总结

术语
术语 解释 举例
关键字 在JavaScrip有特殊意义的词汇 let、var、function、if、else、switch、case、break
保留字 在目前JavaScrip中没有意义，但未来可能会具有特殊意义的词汇 int、short、long、char
标识符（标识） 变量名、函数名的另一种叫法 无
表达式 能产生值的代码，一般配合运算符出现 10 + 3、age>=18
语句 一般可执行的代码 if()、for()

基本数据类型和引用数据类型的存储方式

值类型： 简单数据类型/基本数据类型，在储存时变量中储存的是值本身，因此叫做值类型，例如string、number、boolean、undefined、null

引用类型： 复杂数据类型，在储存变量中储存的仅仅是地址（引用），通过new关键字创建对象（系统对象、自定义对象），如object、array、date等

堆栈空间分配区别

栈（操作系统）： 由操作系统自动分配释放存放函数的参数值、局部变量的值等，其操作方式类似于数据结构中的栈，简单数据类型存放到栈里面

堆（操作系统）： 储存复杂类型（对象），一般由程序员分配释放，若程序员不释放，由垃圾回收机制回收，引用数据类型存放到堆里面

不一定准确的说法： 内存可认为分为“堆”和“栈”，简单数据类型储存在栈里面；而复杂数据类型的地址储存在栈里面，内容储存在堆里面，可以通过栈里面的地址找到堆里面的内容
注意，js并没有堆和栈的说法
