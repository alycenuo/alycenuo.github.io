1、声明变量的语法选择

变量声明有三个，分别是var、let、const，var基本不使用，对于后续不改变的变量（例如不会再重新赋值）要使用const，如果不确定使用哪个，优先使用const，后续出错再回来修改

对于会发生改变的简单数据类型不可以用const，但是对于复杂数据类型可以，例如数组、对象，后续要添加删除元素或者属性也仍然可以用const，因为地址没有改变，但如果变更为新数组或对象仍然会报错，新数组、对象的地址是会改变的

2、DOM

DOM（document object model 文档对象模型）是用来呈现以及与任意HTML或XML文档交互的API，通俗易懂的说，DOM是浏览器提供的一套专门用来操作网页内容的功能

作用：开发网页内容特效和实现用户交互

DOM树

  • 将HTML文档的结构以树状呈现，目的是直观展现标签与标签间的关系

DOM对象： 浏览器根据html标签生成的 JS对象（在HTML里面叫标签，在js叫对象）

document是最大的对象，其次是html标签对象

  • 所有的标签属性都可以在这个对象上面找到
  • 修改这个对象的属性会自动映射到标签身上

DOM核心思想： 把网页内容当对象处理

document对象

  • 是DOM里面提供的一个对象
  • 它提供的属性和方法是 用来访问和操作网页内容的，例如document.write()
  • 网页所有内容都在document里面

获取DOM对象的常用方法

跟CSS选择器类似，选中对象才能操作，查找DOM元素就是利用JS选择页面中标签元素
括号里面用引号，不然会识别为变量

选择匹配的第一个元素

语法
document.querySelector('参数')

参数
包含一个或者多个有效的CSS选择器 字符串(因此必须加引号)

返回值
CSS选择器匹配的 第一个元素，一个HTMLElement对象，如果没有匹配到会返回null

选择匹配的多个元素

语法
document.querySelectorAll('参数')

  • 与单个元素时不同，因为返回的是关于对象的数组，所以不能直接操作修改，一般通过遍历的方式修改

querySelectorAll生成的是伪数组，有长度和索引号，但没有push()、pop()等数组方法

哪怕只有一个元素，通过querySelectAll()获取过来的也是一个伪数组

3、操作元素内容

  • DOM对象都是根据标签生成的，所以操作标签，本质上就是操作DOM对象
  • 就是操作对象使用的点语法
  • 如果想要修改标签元素的里面的内容，则可以使用如下几种方式：
  1. 对象.innerText 属性： 显示纯文本，不解析标签，将文本内容添加/更新到任意标签位置
  2. 属性对象.innerHTML 属性： 会解析标签，多标签建议使用模版字符，将文本内容添加/更新到任意标签位置

操作元素常用属性

  • 可以通过JS设置/修改标签元素属性，比如通过src更换图片
  • 最常见的属性比如：href、title、src等
