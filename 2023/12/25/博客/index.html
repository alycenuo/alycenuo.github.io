博客搭建

2023/12/25

这是一篇零基础萌新搭建博客的记录，目的是方便自己查阅，所以解释和步骤较为简略。
值得一提的是，虽然博客的主题是ParticleX，不过搭建过程参考的是Fomalhaut的博客搭建文章。在此对两者表示感谢~

一、博客搭建

1、前置安装git和node.js，命令行输入git --version和node --version检验是否安装成功。

2、部署过程
依顺序如下：

git config –global user.email “you@example.com” #github注册邮箱
git config –global user.name “Your Name” #github用户名
ssh-keygen -t rsa -C “xxxxxx@xxx” #生成密钥，””内输入自己github邮箱地址

type: git
repo: 自己新建的github仓库地址
branch: main

npm install hexo-deployer-git –save # 安装插件
hexo clean && hexo generate && hexo deploy
hexo clean; hexo generate; hexo deploy # 上传本地内容

cd themes
git clone https://github.com/theme-particlex/hexo-theme-particlex.git particlex –depth=1

hexo clean
hexo generate
hexo deploy


## 博客写作

1、新建文章

`hexo new 文章名字`即可在_post生成对应包含默认信息的.md文件。

2、文章缩略

`<!-- more -->`后面的内容将不会展示在文章预览页。
